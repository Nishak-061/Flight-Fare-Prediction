Flight Fare Prediction

Overview
This machine learning project predicts flight ticket prices based on airline, source, destination, duration, and other travel details.

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

Machine Learning Algorith
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

Project Steps
- Data Cleaning
- Feature Engeneering
- Exploratory Data Analysis
- Model Training
- Model Evaluation

Result
The model predicts flight fares with good accuracy.
